Application Class

Extends EventDispatcher
Defined in: Application:6
Namespace: cloudkid

Creates a new application, for example (HappyCamel extends Application) manages displays, update loop controlling, handles resizing

var app = new Application({fps:60, resizeElement:window});

      The element to resize the canvas to
