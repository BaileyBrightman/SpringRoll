springroll.Captions Class

Defined in: springroll.Captions:12
Module: Captions

A class that creates captioning for multimedia content. Captions are created from a dictionary of captions and can be played by alias. Captions is a singleton class.

Constructor

springroll.Captions

(
  • [captionDictionary=null]
  • [field=null]
  • [selfUpdate=true]
)

Defined in springroll.Captions:12

Parameters:

  • [captionDictionary=null] Dictionary optional

    The dictionary of captions

  • [field=null] createjs.Text | PIXI.Text | PIXI.BitmapText | DOMElement | String optional

    A text field to use as the output for this captions object, if a string type is supplied, captions will use the DOMElement by id.

  • [selfUpdate=true] Boolean optional

    If the captions should update itself

Example:

    var captionsDictionary = {
                                        "Alias1": [
                                            {"start":0, "end":2000, "content":"Ohh that looks awesome!"}
                                        ],
                                        "Alias2": [
                                            {"start":0, "end":2000, "content":"Love it, absolutely love it!"}
                                        ]
                                    };
                                
                                    // initialize the captions
                                    var captions = new springroll.Captions(captionsDictionary);
                                    captions.play("Alias1");
