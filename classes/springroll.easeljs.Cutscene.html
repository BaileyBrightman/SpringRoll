springroll.easeljs.Cutscene Class

Defined in: springroll.easeljs.Cutscene:17
Module: EaselJS Cutscene

Cutscene is a class for playing a single EaselJS animation synced to a single audio file with springroll.Sound, with optional captions. Utilizes the Tasks module.

Constructor

springroll.easeljs.Cutscene

(
  • options
)

Defined in springroll.easeljs.Cutscene:17

Parameters:

  • options Object

    The runtime specific setup data for the cutscene.

    • display String | Display

      The display or display id of the EaselJSDisplay to draw on.

    • configUrl String

      The url of the json config file describing the cutscene. See the example project.

    • [loadCallback] Function optional

      A function to call when loading is complete.

    • [pathReplaceTarg] String optional

      A string found in the paths of images that should be replaced with another value.

    • [pathReplaceVal] String optional

      The string to use when replacing options.pathReplaceTarg.

    • [imageScale=1] Number optional

      Scaling to apply to all images loaded for the cutscene.

    • [captions] Captions optional

      A Captions instance to display captions text on.
