    The display on which the transition animation is displayed.

  • transition createjs.MovieClip | PIXI.Spine

    The transition MovieClip to play between transitions

  • transitionSounds Object

    Data object with aliases and start times (seconds) for transition in, loop and out sounds: {in:{alias:"myAlias", start:0.2}}. These objects are in the format for Animator from CreateJSDisplay or PixiDisplay, so they can be the alias instead of an object.
